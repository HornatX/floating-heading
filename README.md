## 简体中文

### 核心功能

#### 1. 智能标题追踪
实时检测当前滚动位置对应的标题，自动更新悬浮窗口内容。

#### 2. 自由拖拽定位
窗口位置可自由拖拽，支持锁定位置防止误触。

#### 3. 灵活层级选择
支持 H1-H6 任意标题层级，右键菜单快速切换。

#### 4. 外观自定义
可调节字体大小、窗口宽度、圆角、背景色和字体颜色。

#### 5. 平滑滚动定位
点击悬浮标题可快速跳转到对应位置。

#### 6. Markdown 样式控制
可选择保留或忽略标题中的 Markdown 格式（粗体、链接等）。

***

## 用法

1. 安装插件后，在任意 Markdown 文件中滚动，悬浮窗口会自动显示当前标题。
2. 拖拽窗口到喜欢的位置，右键可锁定/解锁位置。
3. 右键菜单可切换标题层级（H1-H6）。
4. 点击悬浮标题可快速跳转到该标题位置。
5. 在设置中可调整字体大小、颜色等外观选项。

### 安装方法

#### 方法一：社区插件安装（推荐）

待插件通过审核并上架社区市场后：
1. 打开 Obsidian **设置** > **社区插件** > **浏览**。
2. 搜索并选择 `Floating Heading`。
3. 点击 **安装** 并选择 **启用**。

#### 方法二：手动安装

1. 前往 [Releases](https://github.com/hornat/floating-heading/releases) 页面下载最新的 `main.js`、`manifest.json` 和 `styles.css` 文件。
2. 打开您的 Obsidian 库所在的本地文件夹。
3. 进入 `.obsidian/plugins/` 目录，并创建一个名为 `floating-heading` 的文件夹。
4. 将下载的三个文件放入该文件夹中。
5. 在 Obsidian **设置** > **社区插件** 中重新加载并开启该插件。

## English

**Floating Heading** — A draggable floating window for Obsidian that displays the current heading based on scroll position. Stay oriented in long documents with **instant heading visibility, customizable appearance, and smooth navigation**.

***

### Features

#### 1. Smart Heading Tracking
Real-time detection of the heading at your current scroll position, automatically updates the floating window.

#### 2. Free Drag & Position
Drag the window anywhere, lock position to prevent accidental moves.

#### 3. Flexible Level Selection
Support H1-H6 heading levels, quick switch via right-click menu.

#### 4. Appearance Customization
Adjust font size, window width, border radius, background and text colors.

#### 5. Smooth Scroll Navigation
Click the floating heading to jump directly to that position.

#### 6. Markdown Style Control
Choose to preserve or ignore Markdown formatting (bold, links, etc.) in headings.

***

## Usage

1. After installing, scroll through any Markdown file and the floating window will automatically display the current heading.
2. Drag the window to your preferred position, right-click to lock/unlock.
3. Right-click menu allows switching heading levels (H1-H6).
4. Click the floating heading to jump to that position.
5. Customize font size, colors and other appearance in settings.

***

### Installation

#### Method 1: Community Plugins (Recommended)

Once the plugin is reviewed and listed on the community marketplace:
1. Open Obsidian **Settings** > **Community plugins** > **Browse**.
2. Search for and select `Floating Heading`.
3. Click **Install** and then **Enable**.

#### Method 2: Manual Installation

1. Go to the [Releases](https://github.com/hornat/floating-heading/releases) page to download the latest `main.js`, `manifest.json` and `styles.css` files.
2. Open your Obsidian vault folder on your computer.
3. Navigate to the `.obsidian/plugins/` directory and create a folder named `floating-heading`.
4. Place the downloaded files into this folder.
5. Reload and enable the plugin in Obsidian **Settings** > **Community plugins**.

***

### Hotkeys

You can set hotkeys in **Settings** > **Hotkeys** for:
- **Toggle floating heading visibility**: Quickly show/hide the floating heading window
